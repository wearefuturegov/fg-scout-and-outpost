Scout could be easily updated to include the CQC widget. The [only information the widget requires is the](https://www.google.com/url?q=https://www.cqc.org.uk/guidance-providers/ratings/using-cqc-widget-troubleshooting&sa=D&source=editors&ust=1674242444616814&usg=AOvVaw3_Hz-NeRQ6UqmG7S_EKaPk) [LocationID](https://www.google.com/url?q=https://www.cqc.org.uk/guidance-providers/ratings/using-cqc-widget-troubleshooting&sa=D&source=editors&ust=1674242444617000&usg=AOvVaw3sTC47Xhg1xqI7bijLOhWL)Â which can be added as a custom field to a service in Outpost.

The CQC widget could also be included on any other website where you can add your own code and the data from Outpost.
